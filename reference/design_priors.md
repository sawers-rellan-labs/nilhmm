# Single-locus genotype-frequency priors for a breeding design

Single-locus genotype-frequency priors for a breeding design

## Usage

``` r
design_priors(design)
```

## Arguments

- design:

  Design key (e.g. `"BC2S2"`, `"BC2S3"`). Interim hardcoded values; S6
  will move these into the bundled `breeding_designs` dataset
  (regenerated by `data-raw/make_breeding_designs.R`).

## Value

`list(g, f_1, f_2)`.

## Examples

``` r
design_priors("BC2S2")   # bulked skim
#> $g
#> [1] "BC2S2"
#> 
#> $f_1
#> [1] 0.0625
#> 
#> $f_2
#> [1] 0.0938
#> 
design_priors("BC2S3")   # BRB
#> $g
#> [1] "BC2S3"
#> 
#> $f_1
#> [1] 0.0312
#> 
#> $f_2
#> [1] 0.1094
#> 
```
