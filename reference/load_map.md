# Load the bundled consensus map

Load the bundled consensus map

## Usage

``` r
load_map(version = "v5")
```

## Arguments

- version:

  Map version key (default `"v5"`); version-namespaced and overridable.

## Value

A map data.frame (marker -\> chr, cM, bp).

if (FALSE) { # \dontrun{
